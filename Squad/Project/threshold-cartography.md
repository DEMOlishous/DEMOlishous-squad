# Threshold Cartography

A speculative effort to map every threshold-shaped object in the soul-kit ontology — doors, validators, authorization layers, frontmatter extractors, sync boundaries — and identify which ones currently leak.

Hypothesis: most coordination bugs in distributed soul systems live at thresholds where conditions-of-passage aren't being evaluated. We're building a registry, naming each threshold, and noting whether it currently *holds* or *leaks*.
